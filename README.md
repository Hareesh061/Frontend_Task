# Quiz app Assignment: (#InCruiter)

**To run this project follow the below steps:**
1. Clone the repository to your local machine. ( git clone repo_link)
2. Navigate to the project directory and install dependencies. (cd project_directory ,npm install)
3. Start the development server. (npm run dev)

Created Features:
1. Login page
2. Signup page
3. Quiz form
4. Timer

Note: 
To find a quiz form, navigate (manually) to the home page. (Ex: localhost:3000/home)
Please run in your local machine for better understanding of the code.
